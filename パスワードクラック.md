# パスワードクラック

## 暴力破解（Brute Force Attack）

暴力破解也叫总当たり攻击。

这是指尝试所有可能的字符串组合来破解密码的攻击方式。

`Brute Force` 原意有“靠蛮力”“强行尝试”的意思。

## 反向暴力破解攻击

反向暴力破解攻击是指，先选定一个可能被用作密码的字符串，将这个密码固定下来，然后针对多个不同的 ID 逐一尝试登录的攻击方式。

通常的暴力破解是针对同一个 ID 不断尝试不同密码，因此在多次失败后，常常会被“密码锁定”机制拦截。

但反向暴力破解是把同一个密码用于多个账号逐个尝试，所以单纯依靠密码锁定的对策往往难以发挥效果。

它与密码喷洒攻击（Password Spraying）基本上是同一类思路，也就是将少数常见密码尝试用于大量账号。

## 字典攻击

字典攻击是指攻击者准备一个专门的“字典”列表，然后把字典中收录的单词、常见字符串或常用密码，从头到尾依次作为密码进行尝试的攻击方式。

### 对策

- 不要把字典中常见的单词、术语用作密码
- 不要使用容易被猜到的字符串作为密码，例如 `123456`、`password`、`qwerty` 等

## 类推攻击

类推攻击是指攻击者根据该账户用户的个人信息等线索，推测其可能使用的密码并进行尝试的攻击方式。

### 对策

- 不要把生日、住址、电话号码等个人信息作为密码使用

## 凭证填充攻击

凭证填充攻击是指，攻击者利用某个网站或系统中泄露出来的 ID 和密码列表，去尝试登录其他网站或系统的一种非法登录攻击。

这种攻击主要针对用户重复使用密码的情况，因此也可以看作是一种“密码列表攻击”。

其中，利用机器人实施的大规模自动化凭证填充攻击尤其常见。

## 彩虹表攻击

服务器等系统在保存密码时，通常不会直接保存密码原文，而是将其转换为哈希值后再保存。

彩虹表攻击是指，攻击者预先准备一张“常见密码字符串及其对应哈希值”的对照表，也就是彩虹表，然后根据获得的哈希值反推出原始密码的攻击方式。

它也常被称为“彩虹表攻击”。

## 重放攻击

重放攻击是指，攻击者在网络上窃听通信内容后，将截获到的数据（例如 ID 和密码）再次发送，从而实施非法操作（如非法登录）的一种攻击手法。

### 对策

- 使用一次性密码
- 使用时间戳等机制，防止旧数据被重复利用

## 针对密码破解的整体对策

- 引入多因素认证
- 引入无密码认证
- 在哈希前加入随机字符串（盐值，Salt）
- 制定强有力的密码策略
- 使用密码管理器
- 检查密码是否属于已泄露密码
