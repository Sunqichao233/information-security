# 脅威・脆弱性・対策

## 脅威・脆弱性・対策の3つのグループ

1. 人的グループ
2. 技術的グループ
3. 物理的グループ

## 1. 人的グループ（人的组）

- 脅威（威胁）:
  ソーシャルエンジニアリング、BEC、人為的ミス（誤操作、アクセス権の誤設定、情報端末の紛失等）
  
  中文：社会工程、BEC（商业邮件诈骗）、人为失误（误操作、访问权限配置错误、信息终端丢失等）

- 脆弱性（脆弱性）:
  低い意識、シャドーIT等
  
  中文：安全意识薄弱、影子 IT 等

- 対策（对策）:
  ガイドライン策定、啓蒙、教育等
  
  中文：制定指南、宣传启发、教育培训等

### 补充说明

#### 社会工程

不使用技术手段，而是利用人的心理弱点来获取信息或滥用信息的行为。

例子：

- 欺骗他人，套取 ID 和密码
- 翻找办公室内的垃圾箱，获取重要信息
- 偷看离席员工桌面上的资料或电脑画面

#### 商业邮件诈骗（BEC）

冒充公司高层或交易对象，通过精心伪装的邮件欺骗相关负责人，从而窃取信息，或诱导其将现金转入伪造账户的诈骗行为。

## 2. 技术组

- 威胁：
  网络攻击、非法入侵、恶意软件、系统错误等

- 脆弱性：
  程序缺陷、安全漏洞

- 对策：
  各种安全实现技术、安全产品等

### 补充说明

#### 非法入侵

通过不正当手段侵入信息系统，并对信息进行破坏、篡改等行为。

#### 安全漏洞

计算机系统、网络、软件等在安全方面存在的弱点或缺陷。

## 3. 物理组

- 威胁：
  自然灾害、火灾、事故、盗窃、破坏、非法闯入

- 脆弱性：
  不符合现代标准的老旧建筑、管理不足

- 对策：
  抗震设备、计划性修缮、上锁管理、出入室管理等

## 不正行为发生的机制与防范机制

### 不正三角形

人产生不正行为的机制。当以下三个条件同时具备时，更容易发生不正行为：

- 导致不正行为发生的处境
- 容易实施不正行为的环境
- 将不正行为正当化的理由

### 情境犯罪预防

通过建设不容易发生犯罪和不正行为的环境，来预防其发生的思路。

### 破窗理论

如果空房子的窗玻璃被打破后长期放任不管，其他窗户也会继续被打破，垃圾也会被随意丢弃，整体环境风气会不断恶化，最终使整栋房屋陷入更糟的状态。

由此引申出的观点是：即使是轻微的违规或犯罪行为，也应当及时、妥善地加以处理，这样可以防止秩序和道德水平下降，从而实现预防犯罪的效果。

### 防犯环境设计

为创造不易发生犯罪的环境，除了依靠人的防犯活动这一软件层面的措施外，还要对建筑、道路、公园等物理环境这一硬件层面进行整备与强化，从而形成不易发生犯罪的环境。

其直接方法包括：

- 强化目标物
- 控制接近

其间接方法包括：

- 确保监视性
- 确保领域性

关键在于将这些方法综合组合起来运用。

## 攻击者的种类

常见的攻击者类型包括：

- Cracker
- Script Kiddie
- Bot Herder
- 内部人员作案者
- 愉快犯
- 诈骗犯
- 故意犯
- 暗网相关人员等

### Cracker

Cracker 是指黑客中带有恶意并实施攻击的人。

需要注意的是，Hacker 原本只是“精通计算机技术的人”的意思，本身并不带有善恶含义。

### Script Kiddie

Script Kiddie 是指只是利用网络上的攻击工具或脚本，在并不具备特别高深技术知识的情况下实施非法访问或攻击的人。

这类人的动机往往偏向玩乐、逞强或出风头等带有兴趣性质的目的。

其中 `Kiddie` 带有“像小孩子的”“幼稚”的贬义色彩。

### Bot Herder

Bot Herder 是指利用 C&C 服务器远程操控僵尸程序，构建和管理僵尸网络，并利用其实施网络攻击的人。

### 暗网

暗网是指无法通过一般搜索引擎检索、且匿名性较高的一类互联网内容总称。

由于其匿名性强，也可能被用于网络攻击、非法商品交易等违法活动。

在访问暗网时，有时需要使用特殊的软件。近年来，也存在在实施攻击时利用暗网的情况。

## 攻击者的动机

常见动机包括：

- 金钱获取
- 黑客行动主义
- 网络恐怖主义等

### 黑客行动主义

黑客行动主义是指为了政治性或社会性的主张、思想，而实施黑客攻击、非法入侵等网络攻击行为。

这个词由 `Hack` 和 `Activism（行动主义）` 组合而成。
