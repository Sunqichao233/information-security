# 秘匿化・匿名化、脆弱性管理

## 秘匿化

使信息从外部无法被看见（即保持机密）。通常会使用加密、恰当的访问控制等手段来实现。

## 匿名化

通过删除或替换数据中可识别个人、法人或组织的信息，使其无法识别到特定对象。这在个人信息保护方面尤其重要。

常见的匿名化方法包括：字段删除、记录删除、单元格删除，以及顶部（底部）编码等。
